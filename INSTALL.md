# 安裝依賴套件

執行以下命令安裝所需套件：

```bash
pnpm add -w @supabase/supabase-js react-markdown remark-gfm rehype-highlight rehype-raw date-fns highlight.js
```

## 套件說明

- `@supabase/supabase-js`: Supabase 客戶端
- `react-markdown`: Markdown 渲染
- `remark-gfm`: GitHub Flavored Markdown 支援
- `rehype-highlight`: 程式碼高亮
- `rehype-raw`: 支援 HTML 標籤
- `date-fns`: 日期格式化
- `highlight.js`: 程式碼高亮樣式
